# Customer Churn Prediction

## Project Overview
This project predicts whether a customer is likely to churn using machine learning techniques.

## Dataset
Customer-Churn-Records.csv
Source: Kaggle.com
https://www.kaggle.com/datasets/radheshyamkollipara/bank-customer-churn

## Model Used
Logistic Regression

## Steps to Run the Project
1. Clone or download the repository
2. Open the Python file or Jupyter Notebook
3. Install required libraries if needed
4. Run all cells / execute the script
5. Provide user input when prompted to get churn prediction

## Results
The model achieved approximately ~99% accuracy on the test dataset with very high precision and recall. It successfully predicts customer churn and provides churn probability for user-given inputs.
